02

安全性

六层纵深防御体系 — 安全靠架构硬执行，不靠文字指令软约束

六层纵深防御体系

1
Docker 沙箱隔离 容器 uid=2000 + 独立网络 172.30.0.0/16 + iptables 公网封锁 + workspace 挂载隔离
2
用户命名空间隔离 ent_{userId}_{agentName} 三层命名空间，数据物理隔离
3
多层权限管控 三维度白名单（工具 / 技能 / MCP），提示词隐藏 + 执行层双保险校验
4
企业级审计系统 双写日志（DB + JSONL）+ HMAC-SHA256 签名链防篡改，覆盖 20 个操作点
5
认证授权 JWT + LDAP/AD 集成 + Token 刷新 + 暴力破解防护 + 已删用户即时拦截
6
AI 行为熔断 三级阈值（警告8 / 拒绝15 / 终止25）+ MCP 连续失败熔断 + 30分钟超时兜底

第一层：Docker 沙箱隔离

AI Agent 需要执行代码、操作文件，如何防止越权访问和数据泄露？

👤 用户隔离

容器内 uid=2000（非宿主机用户 uid=1000），防止 bind mount 权限穿透

即使容器逃逸也无法读取宿主机敏感文件

🌐 网络隔离

独立 Docker 网络 + iptables 规则

  • 允许：企业内网
  • 封锁：互联网出站
  • 封锁：MySQL/Redis/网关端口

📁 文件隔离

仅挂载用户专属 workspace 目录

workspaceOnly=true 强制限制，每个用户/Agent 只能读写自己的工作空间

第二 ~ 三层：命名空间隔离 & 权限管控

命名空间隔离

Agent ID: ent_{userId}_{agentName}
Session:  agent:ent_{userId}_{agentName}:session:{id}
目录:   data/users/{userId}/workspace/
  • Agent 配置：用户只能管理自己的 Agent
  • 会话数据：用户只能查看自己的对话
  • 文件存储：每个用户独立工作空间
  • AI 记忆：agentAccess 白名单控制
  • 所有权校验：startsWith 前缀匹配

三维度白名单权限

维度控制对象
工具权限文件读写、命令执行
技能权限可调用的企业技能
MCP 权限外部工具 / 数据库连接

双保险设计

1. 软限制：提示词层隐藏未授权工具

2. 硬限制：执行层再次校验并拒绝

第四 ~ 六层：审计、认证、熔断

📝 审计系统

双写策略

  • MySQL 实时存储 → 即时查询
  • JSONL 按天归档 → 离线分析

防篡改

HMAC-SHA256 签名链，每条记录包含前一条哈希，删除/篡改即断裂

覆盖 20 个关键操作点

🔒 认证授权

企业认证JWT + LDAP/AD
Token 安全2h 过期 + 7d 刷新
暴力破解10次/分钟 → 告警
已删用户DB 实时校验拦截
环境变量白名单注入

⚡ AI 行为熔断

三层熔断防护

层级阈值动作
警告8 次发送警告
拒绝15 次拒绝调用
终止25 次强制终止

+ MCP 连续失败 3 次熔断
+ 30 分钟执行超时兜底

让 AI 能操作企业内部系统，从"对话"升级为"执行"

管理员定义 MCP Server
→
平台自动发现工具

AI 安全调用

三种运行模式

模式场景隔离
宿主机直连企业受信基础设施环境变量白名单
Docker 沙箱用户自定义工具完整容器隔离
HTTP 远程第三方服务网络级隔离

企业级增强

  • Agent 级 MCP 权限白名单
  • 数据库凭证 AES-256-GCM 加密
  • 环境变量不继承全量 process.env
  • 工具连续失败 3 次自动熔断
  • 两级缓存（磁盘 + 内存）

🎯 企业技能系统 (Skill)

将复杂的专业流程封装为"一键可用"的 AI 技能

解决的问题：报表生成、数据分析等标准化流程，靠 AI 每次从头理解效率低、质量不稳定

技能包结构

skill-package/
  SKILL.md     # AI 操作指南
  scripts/main.py  # 执行脚本
  examples/     # 示例数据
  templates/    # 报表/PPT 模板

两种管理模式

类型管理方隔离
企业技能管理员统一部署引擎原生发现
个人技能用户自行上传Docker 沙箱

安全保障

  • 所有技能在 Docker 容器中执行
  • 共享 Python 虚拟环境（预装常用库）
  • 脚本只能读写用户 workspace
  • 结果自动输出，用户可直接下载

业务场景

  • "帮我分析这份销售数据" → 自动 ECharts 可视化报表
  • "把这份报告做成 PPT" → 直接输出成品文件
  • 管理员部署新技能 → 全企业即刻可用

💡 心跳巡检监控 & 💬 多渠道 IM 集成
