## 配置 JSON 预览

> 以下 JSON 将同时写入 `chart_config.json` 文件，供生成脚本直接使用。

```json
{
  "title": "销售数据分析仪表盘",
  "charts": [
    {"type": "kpi", "title": "总销售额", "measures": ["销售额"], "config": {"agg": "sum"}, "col_span": 3},
    {"type": "kpi", "title": "订单数量", "measures": ["订单ID"], "config": {"agg": "count"}, "col_span": 3},
    {"type": "gauge", "title": "目标达成率", "measures": ["达成率"], "config": {"max": 100, "unit": "%"}, "col_span": 3},
    {"type": "kpi", "title": "平均客单价", "measures": ["销售额"], "config": {"agg": "avg"}, "col_span": 3},
    {"type": "line", "title": "月度销售趋势", "dimensions": ["日期"], "measures": ["销售额", "销售量"], "config": {"smooth": true}, "col_span": 12},
    {"type": "bar", "title": "地区销售对比", "dimensions": ["地区"], "measures": ["销售额"], "config": {"horizontal": true}, "col_span": 6},
    {"type": "rose", "title": "产品销售占比", "dimensions": ["产品"], "measures": ["销售额"], "col_span": 6},
    {"type": "funnel", "title": "销售转化漏斗", "dimensions": ["阶段"], "measures": ["用户数"], "col_span": 6},
    {"type": "heatmap", "title": "地区×产品热力图", "dimensions": ["地区", "产品"], "measures": ["销售额"], "col_span": 6}
  ]
}
```

### 配置字段说明

| JSON 字段 | 含义 | 示例 |
|----------|-----|-----|
| `col_span` | 占几列宽度（12 为整行） | `3` = 四分之一行，`6` = 半行 |
| `agg` | 聚合方式 | `sum`（求和）、`avg`（平均值）、`count`（计数）、`max`（最大值）、`min`（最小值） |
| `measures` | 度量指标（用于计算的数值字段） | `["销售额", "销售量"]` |
| `dimensions` | 分析维度（用于分组的分类字段） | `["日期"]`、`["地区", "产品"]` |
| `smooth` | 是否启用平滑曲线 | `true` / `false` |
| `horizontal` | 是否水平方向显示 | `true` = 条形图，`false` = 柱状图 |
| `top_n` | 仅保留前 N 项，其余归并 | `10` |
| `others_label` | 归并后的显示名称 | `"其他"` |
